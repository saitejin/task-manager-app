# 📝 Task Manager App

A simple and responsive task management app built using HTML, CSS, and AngularJS.

## 🚀 Features

- Add, delete, and complete tasks
- Priority tagging (High, Medium, Low)
- Data persistence using `localStorage`
- Clean and minimal UI

## 💻 Technologies Used

- HTML
- CSS
- AngularJS
